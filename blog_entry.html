Einleitung

Damit nun tatsächlich auch das zuletzt erworbene Wissen über Java angewandt wird, habe ich mir überlegt ein einfaches Tool aus dem Pentesting-Bereich zu programmieren: ein Port Scanner. Das Programm soll zwar bei weitem nicht die Fähigkeiten von bereits ethablierten Tools wie nmap oder RustScan besitzen, dennoch kann somit auch eine kleine Einführung in sockets erfolgen.

PortScanner.java

Innerhalb der PortScanner.java-Datei muss zu Beginn wieder geprüft werden, ob ein Argument bei der Programmausführung auch mitgeliefert wurde, da so der Benutzer die Ziel-Adresse angeben soll: 

if (args.length < 1) {
    error("Too few arguments! You have to enter the host-IP!");
    return;
}
String host_ip = args[0], service_name = "";
int host_port = 0, open_ports = 0;
Boolean status = ServiceGrabber.get_all_services();
if (status == false) {
    return;
